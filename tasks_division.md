## Shared Responsibilities

The following responsibilities are collectively managed by all team members:

- **Version Control Management** – Consistent use of Git, proper commit practices, feature branching strategies, and pull request protocols
- **Quality Assurance** – Comprehensive testing, debugging, and validation of implemented features
- **Data Modeling** – Entity-Relationship (ER) Diagram design and refinement
- **Individual Reflection** – Personal contributions and learning outcomes documentation
- **Report Review** – Collaborative review and refinement of final project documentation
